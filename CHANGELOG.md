# 更新记录

本项目遵循语义化版本。

## 未发布

### 新增

- ABI v2 原生 SQLite 后端与固定 bundled SQLite；
- macOS、Linux、Windows 的 x86-64 与 ARM64 制品；
- 文件、内存、临时、URI、只读、WAL、外键和 busy timeout 打开配置；
- 真正预编译语句资源、参数数、显式关闭与父连接级联；
- 交互式事务、立即事务和保存点；
- 表、列、索引、索引列和外键反射；
- 迁移状态、检查、干运行、SHA-256 校验和和旧记录采用；
- 规范言据文本、JSON1 原生结构与参数化路径条件；
- 无文件权限、原生和 CLI 三个独立消费者。

### 变更

- `打开` 从 `sqlite3` CLI 改为原生生产后端；
- CLI 改为 `打开CLI`、`打开为` 的显式兼容路径；
- 迁移在原生后端使用立即写锁、锁内重规划和原子登记；
- CLI 占位符扫描支持方括号标识、转义反引号及行/块注释。

### 安全

- 文件 URI 拒绝 authority、百分号编码、片段、多个查询串和重复 mode；
- 内存 URI 只在无歧义时跳过文件权限；
- 原生制品固定目标、普通文件、大小和 SHA-256；
- Windows 制品静态链接 CRT，macOS 使用稳定安装名与签名。

## 0.1.0

- 首次发布 CLI 连接、安全参数绑定、预编译语句外观、原子事务批、保存点和基础迁移。
