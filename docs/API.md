# 言库·轻舟 API

## 连接

- `打开(数据库路径)`：使用 `sqlite3` 与 30 秒超时；
- `打开为(数据库路径, 程序, 超时毫秒)`：自定义 CLI；
- `以执行器(法)`：注入符合言库请求/响应约定的执行器；
- `执行器工厂(配置)`：可直接传给 `言库.SQLite驱动`。

`SQLite连接` 提供 `执行`、`查询`、`预编译`、`开始事务`、`立即事务`、`批量执行`、`安全执行`、`关闭`。`批量执行` 接受 SQL 文字或 `{SQL, 参数}` 典列，并在一个立即事务中提交。

## 参数与语句

`绑定(SQL, 参数)` 只替换普通 SQL 上下文中的 `?`，不碰单引号文字、双引号标识和反引号标识。文字使用 SQLite 单引号双写规则；列和典以 JSON 文字绑定。

`SQLite预编译` 保存 SQL 模板，提供 `执行`、`查询` 与用于审计的 `已绑定`。

## 事务批

`事务批` 提供 `执行`、`保存点`、`回滚至`、`释放点`、`脚本`、`提交`、`回滚`。提交前不会改动数据库；提交使用一个 SQLite CLI 进程执行完整脚本。

## 迁移

`迁移(版本号, 名称, 升级SQL, 降级SQL)` 描述迁移。`默认迁移器(连接)` 使用 `_yanxu_migrations` 表，提供 `已应用`、`待升级`、`升级`、`回退`。
